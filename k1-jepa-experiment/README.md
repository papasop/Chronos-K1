# k1-jepa-experiment

Exploratory application track for K=1 constraints in JEPA-style representation learning.

This directory is intentionally separate from `k1-manifold-core`. The core package is the publishable mathematical substrate; this track is an experiment log and sandbox for representation-learning ideas.

## Planned Work

- `01_baseline_lejepa.ipynb`: baseline LeJEPA reproduction.
- `02_k1_constraints_v1.ipynb`: first K=1 projection experiments.
- `03_symplectic_loss.ipynb`: J_G consistency losses.
- `04_reacher_control.ipynb`: control toy problem.
- `05_identifiability_analysis.ipynb`: identifiability metrics.

## Commands

```bash
make baseline
make k1_v1
make compare
```
